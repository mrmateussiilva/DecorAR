# DecorAR

Base de um SaaS em Next.js 14 para upload de imagem e preview em um plano 3D texturizado com React Three Fiber.

## Stack

- Next.js 14 (App Router)
- TypeScript
- TailwindCSS
- React Three Fiber
- `@react-three/drei`
- Estrutura preparada para WebXR (sem fluxo XR implementado)

## Como rodar

```bash
pnpm install
pnpm dev
```

Abra `http://localhost:3000/ar`.
